# Radix UI Component Dashboard

Um dashboard moderno e acessível construído com **React**, **TypeScript** e **Radix UI**. Este projeto foca em demonstrar a implementação de componentes de interface robustos, totalmente estilizados para **Dark Mode** e seguindo as melhores práticas de acessibilidade. O projeto está em desenvolvimento onde frequentemente a interface será mudada para melhorar a acessibilidade e para acrescentar novos componentes.

## Funcionalidades

-  **Dark Mode Nativo**: Estilização personalizada em tons de Slate e Indigo.
-  **Acessibilidade**: Componentes baseados no Radix UI, garantindo suporte a teclado e leitores de tela.
-  **Componentização**: Estrutura organizada e reutilizável.
-  **Tipagem Forte**: Desenvolvimento 100% em TypeScript.

##  Tecnologias Utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Radix UI](https://www.radix-ui.com/) (Primitives)
- [Lucide React / Radix Icons](https://lucide.dev/) (Ícones)
- [CSS Modules / Vanilla CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS)

##  Componentes Implementados

O dashboard reúne uma variedade de componentes essenciais:
- **Navegação**: MenuBar.
- **Formulários**: Input, Select, Checkbox, Switch, Slider, RadioGroup.
- **Exibição**: Accordion, Dialog (Modal), Popover.
- **Feedback**: Toast (Notificações).

##  Como Executar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/project-react-radix.git
   ```
2. Instale as dependências:
   ```bash
    npm install ou
    yarn install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
     npm run dev
   ```
4. Acesse no navegador:
   ```bash
    http://localhost:5173
   ```
